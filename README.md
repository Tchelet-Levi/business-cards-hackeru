# Business Card project for HackerU

This repository contains the source code for the Business Card project assignment for HackerU.
It is written in Typescript and uses the following libraries ( among others ) :

###### Front:

- React
- React Redux
- React Router (v6)
- React Helmet
- Zod
- Material UI (MUI)

###### Back:

- JOI
- Bcrypt
- ExpressJS
- Mongoose
- JsonWebToken
